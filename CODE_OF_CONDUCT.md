# Contributor Covenant Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) Code of Conduct, version 2.1.

The full text is available at: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

If you experience or witness unacceptable behavior, please report it by contacting the project team at **ojs@example.com**.

All reports will be reviewed and investigated promptly and fairly.
